# 📖 [The Morse Code] <a name="about-project"></a>


**[The Morse Code]** is a algorithm which can convert any morse code into the readable english message.

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.


### Setup

Clone this repository to your desired folder:


```sh
  cd morse_code
  git clone https://github.com/addisalemseifu/morse_code.git
```

### Install

Install this project with:

```sh
  cd morse_code
  gem install
```
